# AegisMark(小印） 技術支援與隱私說明

最後更新：2026-06-07

## 技術支援

AegisMark(小印） 是一款面向 macOS 的螢幕浮水印工具，支援多圖層浮水印、動態變數、QR 碼、依 App/網站/網路條件顯示、防裁切抖動、視窗遮罩以及截圖浮水印處理。

如需技術支援，請透過 App Store 頁面中的「App 支援」連結聯絡開發者，或提供以下資訊以便排查：

- macOS 版本與裝置型號
- AegisMark 版本
- 問題重現步驟
- 已啟用的權限或相關功能

## 隱私說明

AegisMark(小印）的核心功能在本機完成，不會將浮水印範本、螢幕內容、視窗標題、瀏覽器 URL、截圖檔案或裝置資訊上傳到開發者伺服器。

### 本機儲存

應用程式會在沙盒中保存浮水印圖層、樣式、版面、顯示規則、自訂變數、快速鍵、截圖資料夾安全書籤以及購買狀態。這些資料用於恢復設定和提供功能，預設僅保存在使用者裝置本機。

### 權限用途

- 螢幕錄製：用於視窗遮罩、截圖浮水印處理以及螢幕擷取狀態判斷。
- 輔助使用：用於偵測目前 App、視窗標題和視窗區域。
- 瀏覽器自動化：用於讀取目前瀏覽器 URL，以執行網站網域規則。
- 檔案存取：用於讀取使用者選擇的圖片浮水印，或處理使用者選擇資料夾中的截圖。

### 第三方服務與資料共享

AegisMark(小印）使用 Apple StoreKit 處理訂閱、恢復購買和終身解鎖。付款流程由 Apple 管理，開發者不會直接接觸付款資訊。AegisMark 不出售使用者資料，不用於廣告追蹤，也不主動與第三方共享本機設定或螢幕內容。
