# AegisMark Support and Privacy Notice

Last updated: 2026-06-07

## Support

AegisMark is a macOS screen watermarking utility with multi-layer watermarks, dynamic variables, QR codes, app/website/network rules, anti-crop jitter, window masking, and screenshot watermarking.

For support, use the App Support link on the App Store page or provide:

- macOS version and device model
- AegisMark version
- Steps to reproduce the issue
- Enabled permissions or related features

## Privacy Notice

AegisMark performs its core features locally on your Mac. It does not upload watermark templates, screen contents, window titles, browser URLs, screenshot files, or device information to developer servers.

### Local Storage

The app stores watermark layers, styles, layouts, display rules, custom variables, keyboard shortcuts, screenshot folder security bookmarks, and purchase status in the app sandbox. This data is used to restore settings and provide app features, and it remains on the user's device by default.

### Permissions

- Screen Recording: used for window masking, screenshot watermarking, and screen capture context checks.
- Accessibility: used to detect the active app, window title, and window region.
- Browser Automation: used to read the active browser URL for website domain rules.
- File Access: used to read user-selected watermark images or process screenshots in a user-selected folder.

### Third-Party Services and Sharing

AegisMark uses Apple StoreKit for subscriptions, purchase restoration, and lifetime unlocks. Apple manages the payment flow, and the developer does not directly access payment information. AegisMark does not sell user data, use it for advertising tracking, or proactively share local settings or screen contents with third parties.
