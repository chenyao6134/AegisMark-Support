# AegisMark（小印）サポートとプライバシーについて

最終更新日：2026-06-07

## サポート

AegisMark（小印） は macOS 向けの画面ウォーターマークツールです。複数レイヤー、動的変数、QR コード、App/Webサイト/ネットワーク条件、切り抜き対策ジッター、ウィンドウマスク、スクリーンショットへのウォーターマーク追加に対応しています。

サポートが必要な場合は、App Store ページの「App サポート」リンクから開発者に連絡してください。問い合わせ時には次の情報を含めると調査しやすくなります。

- macOS のバージョンとデバイスモデル
- AegisMark のバージョン
- 問題の再現手順
- 有効にしている権限または関連機能

## プライバシーについて

AegisMark（小印）の主要機能はユーザーの Mac 上でローカルに処理されます。ウォーターマークテンプレート、画面内容、ウィンドウタイトル、ブラウザ URL、スクリーンショットファイル、デバイス情報を開発者サーバーへアップロードしません。

### ローカル保存

アプリはウォーターマークレイヤー、スタイル、レイアウト、表示ルール、カスタム変数、ショートカット、スクリーンショットフォルダのセキュリティブックマーク、購入状態をアプリサンドボックスに保存します。これらは設定の復元と機能提供のために使用され、既定ではユーザーのデバイス内にのみ保存されます。

### 権限の用途

- 画面収録：ウィンドウマスク、スクリーンショット処理、画面キャプチャ状態の確認に使用します。
- アクセシビリティ：現在の App、ウィンドウタイトル、ウィンドウ領域の検出に使用します。
- ブラウザ自動化：Webサイトドメインルールのために現在のブラウザ URL を読み取ります。
- ファイルアクセス：ユーザーが選択した画像ウォーターマークやフォルダ内のスクリーンショット処理に使用します。

### 第三者サービスと共有

AegisMark（小印） はサブスクリプション、購入の復元、永久アンロックに Apple StoreKit を使用します。支払い処理は Apple が管理し、開発者が支払い情報に直接アクセスすることはありません。AegisMark はユーザーデータを販売せず、広告追跡に使用せず、ローカル設定や画面内容を第三者へ積極的に共有しません。
