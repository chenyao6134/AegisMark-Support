# AegisMark（小印） 技术支持与隐私说明

最后更新：2026-06-07

## 技术支持

AegisMark（小印） 是一款面向 macOS 的屏幕水印工具，支持多图层水印、动态变量、二维码、按 App/网站/网络条件显示、防裁剪抖动、窗口避让以及截图水印处理。

如需技术支持，请在 App Store 页面中的“App 支持”链接联系开发者，或提供以下信息以便排查：

- macOS 版本与设备型号
- AegisMark 版本号
- 问题复现步骤
- 已启用的权限或功能

## 隐私说明

AegisMark（小印） 的核心功能在本机完成，不会将水印模板、屏幕内容、窗口标题、浏览器 URL、截图文件或设备信息上传到开发者服务器。

### 本地存储

应用会在沙盒中保存水印图层、样式、布局、显示规则、自定义变量、快捷键、截图目录安全书签以及购买状态。这些数据用于恢复设置和提供功能，默认仅保存在用户设备本地。

### 权限用途

- 屏幕录制：用于窗口避让、截图水印处理以及屏幕捕获状态判断。
- 辅助功能：用于检测当前 App、窗口标题和窗口区域。
- 浏览器自动化：用于读取当前浏览器 URL，以执行网站域名规则。
- 文件访问：用于读取用户选择的图片水印文件或处理用户选择目录中的截图。

### 第三方服务与数据共享

AegisMark（小印） 使用 Apple StoreKit 处理订阅、恢复购买和终身解锁。购买流程由 Apple 管理，开发者不会直接接触支付信息。AegisMark 不出售用户数据，不用于广告追踪，也不主动与第三方共享本地配置或屏幕内容。
